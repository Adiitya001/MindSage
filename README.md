# MindSage

A sanctuary for the mind, built for the moments between.

## Overview

MindSage is a calm, emotionally supportive platform for reflection, journaling, and personal growth. It provides a private, non-judgmental space for users to explore their thoughts and emotions.

## Features

- **Reflection & Journaling**: Private space for personal thoughts and growth
- **Conversational Support**: Guided conversations with an AI companion
- **Therapist Directory**: Explore licensed therapists when you're ready
- **Community Space**: Share thoughts and reflections in a calm, thoughtful environment
- **Moments Archive**: Look back gently at your journey

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Project Structure

- `/app` - Next.js application pages and routes
- `/components` - Reusable React components
- `/chatbot` - Chatbot logic and agent system
- `/lib` - Utility functions
